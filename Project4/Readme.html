This project uses magic methods or Dunder methods in operators. This project is a 'Roll dice' game using the magic method. A Dice class which includes: - constructor - string magic method - All Comparison magic methods - an add operation magic method - some way to roll it A "Cup of Dice" class which includes: - constructor - A list of dice defined on creation - string magic method - All Comparison magic methods (6 in total listing in slides) - Some way to roll all dice - A main method where that runs the game
